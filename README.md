#### projects i'm proud of

**[portfolio-website](https://github.com/yash113gadia/portfolio-website)** — my 3d animated portfolio with custom cursor, scroll-driven animations, and grain overlays. built with react + three.js. → [live](https://yashgadia.vercel.app)
